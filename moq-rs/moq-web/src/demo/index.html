Tips:

  • You can use CSS to stretch the canvas:
    • `width: 100%` instead of `max-width: 100%`.
    • The HTML width/height is set automatically and cannot be adjusted due to OffscreenCanvas limitations.
  • Use `moq-meet` to watch multiple broadcasts under the same `demo/` prefix.
  • Try restarting `just bbb` and playback will automatically recover.
    • Instead of `just all`, you'll need to run `just relay`, `just web`, and `just bbb` separately.
    • The "newer" (unix timestamp) broadcast is used when there are duplicates.
  • No audio support yet.
